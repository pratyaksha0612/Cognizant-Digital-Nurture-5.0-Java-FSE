# Spring Data JPA & Hibernate

## Overview

This module focuses on integrating **Spring Boot**, **Spring Data JPA**, and **Hibernate** with a **MySQL** database to develop database-driven Java applications. The hands-ons demonstrate ORM concepts, CRUD operations, transaction management, repository abstraction, and query methods using Spring Data JPA.



## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate ORM
- MySQL
- Maven
- VS Code
- Git & GitHub

# Hands-ons Completed

## Hands-on 1 – Spring Data JPA Quick Example

### Objective

Develop a Spring Boot application that connects to a MySQL database using Spring Data JPA and Hibernate to retrieve records from the `country` table.

### Concepts Covered

- Spring Boot Project Setup
- MySQL Configuration
- Entity Mapping
- Repository Layer
- Service Layer
- Dependency Injection
- Hibernate Integration


## Hands-on 2 – Hibernate XML Configuration

### Objective

Understand how Hibernate applications can be configured using XML configuration files.

### Concepts Covered

- Hibernate Framework
- XML Configuration
- Session Factory
- ORM Architecture


## Hands-on 3 – Hibernate Annotation Configuration

### Objective

Understand annotation-based Hibernate configuration and entity mapping.

### Concepts Covered

- JPA Annotations
- Entity Mapping
- Primary Keys
- Column Mapping


## Hands-on 4 – Hibernate Configuration using Java

### Objective

Understand Java-based Hibernate configuration without relying on XML configuration files.

### Concepts Covered

- Java Configuration
- SessionFactory
- Hibernate Properties


## Hands-on 5 – ORM Learn Project

### Objective

Implement a complete CRUD application using Spring Data JPA and Hibernate.

### Features Implemented

- Retrieve all countries
- Find country by code
- Add new country
- Update country
- Delete country
- Search countries using partial name
- Custom exception handling
- Transaction management
- Spring Data JPA Query Methods
