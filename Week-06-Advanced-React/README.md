# Week 06 - Advanced React

## Overview

This week focuses on advanced React concepts, including Context API, form handling, form validation, REST API integration, component testing, and API mocking using Jest. The hands-on exercises demonstrate modern React development practices using Hooks, React Testing Library, Jest, and Axios.

## Exercises

### Exercise 14 - Context API

Implemented React Context API to eliminate prop drilling by sharing theme information across multiple components.

Key concepts covered:

- React Context API
- Context Provider
- useContext Hook
- Theme switching
- Component communication

### Exercise 15 - React Forms

Developed a complaint registration application using controlled form components.

Key concepts covered:

- Controlled Components
- Form State Management
- Event Handling
- Dynamic Reference Number Generation
- Form Submission

### Exercise 16 - Form Validation

Built a user registration form with client-side validation.

Key concepts covered:

- Form Validation
- Controlled Components
- Error Handling
- Password Validation
- Email Validation

### Exercise 17 - Fetch API

Integrated a REST API into a React application to display dynamic user information.

Key concepts covered:

- Fetch API
- useEffect Hook
- useState Hook
- Asynchronous Programming
- REST API Consumption

### Exercise 18 - React Testing

Implemented unit testing for React components using Jest and React Testing Library.

Key concepts covered:

- Jest
- React Testing Library
- Component Testing
- Assertions
- Automated Testing

### Exercise 19 - Jest Mocking

Created mock API tests using Axios and Jest.

Key concepts covered:

- Axios
- Jest Mocking
- Mock API Responses
- Component Testing
- API Testing
